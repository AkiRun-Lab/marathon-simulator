# 🍊 マラソン攻略シミュレーター (Marathon Pace Simulator)

**物理モデル（勾配・風・空気抵抗）** に基づき、世界中のマラソンコースを「一定の出力（パワー）」で走り切るための最適なペース配分を計算するWebアプリケーションです。

VDOTや目標タイムを入力するだけで、コースの起伏や当日の風向きを考慮した「攻めのペース戦略」を誰でも簡単に作成できます。

## 🏃 バージョン情報
**Version**: β0.1 (2025-01-10)

## ✨ 主な特徴
*   **物理ベースの計算**: ジャック・ダニエルズのVDOT理論とMinettiの勾配コストモデルを融合。
*   **世界中のコース対応**: GPXファイルを読み込むことで、あらゆるマラソンコースに対応（愛媛、東京、ボストン、ベルリンなど）。
*   **環境条件の考慮**: 風速・風向き、上り坂での強度設定（Hill Strategy）など、きめ細やかな設定が可能。
*   **高度なデータ処理**: 5m精度のリサンプリングと動的な標高平滑化により、ノイズを除去しつつコース特性を正確に反映。
*   **コース比較機能**: 現在のコンディションで走った場合のタイム差や難易度（Toughness）を他コースと比較可能。
*   **スマホ最適化**: レース当日の移動中やスタート前でも操作しやすい、モバイルファーストなUI設計。

*   `app.py`: アプリケーション本体
*   `data/`: GPXデータおよびVDOTテーブル
*   `lib/`: 計算処理モジュール群
